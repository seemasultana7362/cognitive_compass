# Product Overview

Cognitive Compass is a cognitive load-aware multi-agent AI system that enhances developer productivity by detecting confusion in real-time and providing adaptive code explanations.

## Core Value Proposition

Instead of generating code, Cognitive Compass acts as a thinking companion that helps developers understand code faster through:
- Real-time confusion detection
- Hallucination-free explanations
- 3D code flow visualization
- Persistent team knowledge memory

## Target Users

- Software engineering teams
- Enterprise developers
- Tech startups
- Coding bootcamps
- Students learning large codebases

## Key Differentiators

- First cognitive-aware developer AI system
- Multi-agent architecture (5 specialized agents)
- Focus on understanding over generation
- AWS-native AI stack with latest services
- Real productivity measurement
